# bedarkmode

通过油猴脚本给微博、知乎等网站增加 Dark mode 样式

https://naoyeye.github.io/bedarkmode/

### 使用

脚本都放在 download 目录里了，用哪个下哪个就可以。


### 开发

浏览器里调试好脚本，把 js 文件放进 download 目录里就行了，没什么智能的。

### 网站

本地启动 jekyll 

`bundle exec jekyll serve`



`bundle exec jekyll build`
